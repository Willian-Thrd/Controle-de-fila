# Controle de Filas

Este projeto se trata de um controlador de filas.

Ele foi feito para ajudar na organização da gestão dos pedidos, e também para cumpri-los dentro do prazo
Além disso, ele pode ajudar na visualização dos pedidos e é muito fácil e prático de utilizar.

## Público alvo

O projeto foi desenvolvido principalmente para funcionários de pequenas lanchonetes.
Se o estabelecimento for pequeno ou se não possuir um sistema de gestão de pedidos, esse programa seria perfeito para ele

## Exemplos

Vamos imaginar uma lanchonete pequena. Ela não possui um sistema para gerir seus pedidos. Então, para facilitar a vida dos funcionários, esse programa oferece suporte para:

1. Registrar o pedido;
2. Valor cobrado;
3. Nome do cliente e suas informações;
4. Registar as notas dos pedidos.

## Informações técnicas

O Programa no geral foi programado com base em JavaFX. Onde a interface foi feita na linguagem FXML - Com auxílio de programas gráficos - e a parte lógica do programa foi feita com base em Java. O design do programa como cores dos componentes, letras, formas e tamanhos foram configurados a partir de arquivos CSS.
